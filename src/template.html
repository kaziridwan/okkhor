Converted files keep their formatting; Bijoy font tags are re-pointed to a Unicode Bangla font (Nikosh). Install Nikosh, or switch the font to Kalpurush / Noto Sans Bengali in Word.
Binary .doc files are parsed directly in the browser — OLE container, piece table, character and paragraph formatting, and the font table — then rebuilt as a fresh .docx with the same font-aware Bijoy detection as .docx input. Bold, italic, underline, sizes, alignment, images, and tables all survive — including inline pictures, floating shapes, vector diagrams (EMF/WMF, decompressed in the browser), and real table grids with original column widths and borders. Merged cells are the one remaining gap: they come through as separate cells.
Conversion engine adapted from bijoy2unicode by Md. Jehad (MIT).
